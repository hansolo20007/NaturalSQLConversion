# Natural to Query Translator (NQL)
# Converts English language into a SQL query for use with Microsoft SQL Server Management Studio
# Created by Jackson Baird and Matthew Tamulewicz
#
# A groq API key is needed to run this script. Set the enviornment variable GROQ_API_KEY to your key.
# This can be done in VScode by typing into the terminal $env:GROQ_API_KEY = "<Insert Key Here>"
